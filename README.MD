## Langkah-langkah

### 1. Clone Repositori

```bash
git clone https://github.com/ulsreall/sonic-bot.git
```

```
cd sonic-bot
```

### 2. Instal Dependensi

```bash
npm install
```

### 3. Konfigurasi .env File

buat file .env di root projek

```bash
PRIVATE_KEYS=["your private key 1", "your private key 2"]
```

### 4. Run script

```bash
npm run start
```

\*pastikan udah install node js
